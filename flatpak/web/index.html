Chronicler

Linux (Flatpak)

One-time setup. Adds Chronicler as a Flatpak remote so future updates flow through flatpak update:

flatpak install --user --from $PAGES_URL/chronicler.flatpakref

Or, in GNOME Software / KDE Discover, open the .flatpakref file directly.

Updating

flatpak update pro.chronicler.Chronicler

One-shot install (no auto-updates)

Download the latest Chronicler.flatpak from GitHub Releases and run:

flatpak install --user Chronicler.flatpak
